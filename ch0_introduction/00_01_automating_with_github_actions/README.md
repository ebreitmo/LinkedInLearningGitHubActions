# 00_01 Automating With GitHub Actions

Automation is key to modern software development and GitHub Actions makes it easy to automate building, testing, and deploying, right where your code lives.

In this course, you'll be introduced to GitHub Actions, an automation framework built into the world’s most popular code hosting platform.

By learning GitHub Actions, you can streamline your workflows, save time, and focus on building great software.
